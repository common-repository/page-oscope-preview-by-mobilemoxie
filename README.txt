== Description ==

The Page-oscope Preview by MobileMoxie plugin lets you preview how your page appears on different devices in order to help you understand how they are working for mobile users and desktop users as well as mobile search engine bots!

It is even geo-aware, in case you are modifying content for users in different countries! A must-have for every SEO in validating page code for Mobile-First Indexing.

You can also preview your unpublished pages. That way you can ensure your pages look good on different devices before you publish them.
== Installation ==

1. Go to the 'Add New' plugins screen in your WordPress admin area
1. Search for 'Page-oscope Preview by MobileMoxie'
1. Click 'Install Now' and activate the plugin

For a manual installation via FTP:

1. Upload the `page-oscope-preview-by-mobilemoxie` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
